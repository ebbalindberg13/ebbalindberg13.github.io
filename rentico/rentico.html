Rentico

Overview
Rentico is an app which helps families with kids to advertise and rent stuff from each other. Families with children need to buy an awful lot of stuff such as, clothes, toys, pets, books etc. A smarter and more sustainable solution could be if you could simply rent what you need for a period of time and also let others rent things you have but don’t really need currently. This was an home exam assignment, carried out in the span of 2 days. The task was to create an user interface for an app that allowed families to rent stuff from each other. No user tests were conducted due to limited time.


When
October 2022
Tools
Pen, paper, figma
Methods
Sketching, wireframing, prototyping

Styling

Renting is an excellent solution instead of buying new things. Since the target group are families, I wanted to create an app that was fun and cheerful, but not too childish, so that the parents could browse the app together with their kids. I have designed my app based on the crossplatform components from Flutter that can be found on material.io.

Logo
The name is a combination of the wkeywords “Rent” and “Cooperative”. The logo mark consists of two separate ovals that together form a heart to symbolize the hapiness when people come together and colllaborate.

Colors
I chose to work with orange as my primary color, giving a sense of warmth and joy, blue as the secondary color. The primary color is used for the headlines, the bottom navigation bar and other decorative elements. The secondary color is the interactive elements, such as buttons or similar. A neutural grey color is used when things are disabled or not as important as other elements. Hopefully the colors help create some sort of visual hiearchy and also help the UI components with their meaning.

Typography
For my main headlines used "Squada One" which is a fun and playful font that fits the theme of the app. According to the material design guidelines the headline font should be eye catching. For my secondary headlines and body text I chose "Montserrat" which is a more legible fonts in smaller sizes.

User flow

  1. Choose category
    When the user first enter the app she is taken to the home page where only a rough search isa to overwhelm the user by all the choices. However, if there are options here to filter location and rental period or directly search for an item.
  2. Browse products
    Additional filter options are available. The product most important information is displayed on the cards and the availability have color coded markers to help the user scan for viable options
  3. Details page
    Contact information to renter and action for rental are easily accessible.
  4. Order
    Personal details and order can be managed here.

Reflection

Adhering to all type sof guidelines is an important skill to have when designing interfaces, wheter it be brand guidelines or user interface guidelines. We only had a short amount of time to complete the interface and all the screens (not all the screens are included here). To work effectively, it was important to know the material design guidelines so I didn't have to look every single thing up.
